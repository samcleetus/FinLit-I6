# FinLit I6

A financial literacy mobile game built with Godot 4.3.

## Project Details

- **Engine**: Godot 4.3
- **Platform**: Mobile (iOS/Android)
- **Resolution**: 1170x2532 (Mobile)

## Getting Started

### Prerequisites
- Godot 4.3 or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/finlit-i6.git
cd finlit-i6
```

2. Open the project in Godot:
   - Launch Godot 4.3
   - Click "Open Project"
   - Navigate to the project folder
   - Select `project.godot`

3. Run the game:
   - Press `F5` or click the Play button in the editor

## Project Structure

- **Autoloads/** - Global scripts and managers
- **Scenes/** - Game scenes and UI layouts
- **Scripts/** - Game logic and utilities
- **Systems/** - Game systems (gameplay mechanics, etc.)
- **UI/** - UI-related scripts
- **Resources/** - Game resources and settings
- **Assets/** - Images and other media
- **Data/** - Game data files
